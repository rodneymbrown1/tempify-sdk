templify package

Templify SDK

A toolkit for analyzing, mapping, and generating schemas from DOCX and plaintext sources. Provides a unified API around: - Analysis (detectors, forms, matchers) - Schema building (TemplifySchemaBuilder, SchemaGenerator) - Utilities (mappers, intake, workspace)
