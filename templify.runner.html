templify.runner.build_schema(document_xml_path, extract_dir=None)[source]

Build a Templify schema from a document.xml and (optionally) its extracted DOCX folder.

Parameters:

  • document_xml_path (str) – Path to the document.xml file.

  • extract_dir (str | None) – Path to the extracted DOCX directory (unzipped). Required for some features.

Returns:

The generated Templify schema.

Return type:

dict
